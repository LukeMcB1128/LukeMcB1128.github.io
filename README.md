# Modern Blog

A modern, responsive blog built with Next.js, React, and Tailwind CSS. Supports Markdown and MDX content.

## Features

- Responsive design with Tailwind CSS
- MDX support for rich content
- Static site generation
- SEO-friendly
- Dark mode support
- Pagination for blog posts
- Simple write interface

## Getting Started

First, install the dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build for Production

```bash
npm run build
```

This generates a `out` directory with static files ready for deployment.

## Deployment

This project can be deployed to Vercel, Netlify, or GitHub Pages.

## Project Structure

- `pages/` - Next.js pages
- `components/` - Reusable React components
- `styles/` - Global CSS styles
- `content/blog/` - Blog post files (MDX format)

## Customization

You can customize colors, fonts, and other styling by modifying the Tailwind configuration in `tailwind.config.js`.
